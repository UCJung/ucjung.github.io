Section 0.3 — 실전 예시 ①
시니어 + AI Agent(주니어 역할) 협업

상황

시니어가 설계부터 코딩·문서화·테스트까지 전부 수행 중. 설계에 집중할 수 없는 구조.

역할 분담 원칙
원칙설명
설계는 사람AI Agent는 설계를 "실행"하는 역할
지시는 명확하게주니어에게 "알아서 해"는 위험
결과는 반드시 검증코드 리뷰 없이 배포하지 않는 것과 동일
산출물은 파일로구두 보고 ✕ → 문서 기반 소통 ✓
시니어는 고부가가치 업무(설계·검증)에 집중,
실행 업무를 AI Agent에 위임하여 병목 해소
협업 흐름
① 요구사항 분석 · 설계 시니어 (사람)
② 구현 지시 설계문서 + 스택 + 범위 + 저장경로
③ 코드 구현 + 테스트 + 보고서 AI Agent (주니어 역할)
④ 코드 리뷰 · 검증 시니어 — 보고서 기준 확인
⑤ 릴리즈 판단 · 배포 문제 시 수정 지시 → ③으로

Section 0.4 — 실전 예시 ②
주니어 + AI Agent(선임 역할) 협업
선임이 없는 스타트업 — 주니어가 요구사항 구체화 → 설계 리뷰 → 코드 리뷰를 AI Agent와 수행하며 성장
상황

주니어가 개발 총괄. 설계·기술 판단에 확신 부족. 물어볼 선임이 없어 혼자 결정 → 반복적으로 뒤엎는 구조.

핵심 원칙
원칙설명
AI가 대신 안 한다주니어가 직접 구현, AI는 리뷰·조언
결정은 사람이선택지와 근거 제공, 판단은 주니어 몫
"왜?"를 질문AI의 피드백을 이해하면 학습 가속
과정을 문서로리뷰 결과 축적 = 팀 지식 자산
"대신 해주는 도구"가 아닌 "함께 성장하는 파트너"
설계·리뷰 피드백을 받으며 주니어 역량이 성장
협업 흐름 (8단계)
① 요구사항 초안 작성 주니어가 아는 범위에서 정리
② 요구사항 리뷰 AI — 빠진 항목·모순 지적 + 질문
③ 답변 → 요구사항 확정 주니어 — 질문에 답하며 항목 보완
④ 설계 리뷰 + 대안 제시 AI — 2~3가지 방안, 장단점 비교
⑤ 방향 결정 → 직접 구현 주니어 — 근거를 이해하고 스스로 결정
⑥ 코드 리뷰 + 피드백 AI — 설계 대비 일치·보안·개선점
⑦⑧ 수정·학습 → 배포 판단 역량 성장 사이클

Section 0.5
AI Agent 협업 도입 절차
개인·조직의 부족한 부분을 AI Agent로 채우고, 상호 피드백으로 검증하는 4단계 프로세스
4단계 요약
1
부족한 점을 진단한다 → AI Agent에게 부여할 역할 결정
2
수행 방식을 정의한다 → 해당 역할의 업무 방식을 문서화
3
소단위 업무에 적용하며 개선 → 적용 → 피드백 → 지침 개선 사이클
4
효과 확인 후 범위 확대 → 업무 범위·적용 대상 점진 확장
핵심: "AI Agent에게 무엇을 시킬까"가 아니라
"우리에게 무엇이 부족한가"에서 출발합니다.

Section 0.5 — Step 1 & 2 상세
진단 → 역할 정의 → 수행 방식 문서화
Step 1에서 병목을 파악하고, Step 2에서 AI Agent가 따를 구체적 업무 방식을 정의합니다
Step 1. 부족한 점 진단
시니어 팀

밀리는 업무: 구현·문서화·테스트
원인: 주니어 부재
→ AI = 주니어 역할

주니어 스타트업

밀리는 업무: 설계 리뷰·코드 리뷰
원인: 선임 부재
→ AI = 선임 역할

Step 2. 수행 방식 정의

"주니어처럼 해줘"는 지시가 아닙니다. 해당 역할의 실제 업무 방식을 문서화해야 AI Agent가 의도대로 움직입니다.

수행 방식이 명확할수록 AI Agent의 결과 품질이 올라갑니다.
→ 이 문서가 CLAUDE.md 또는 별도 지침 파일이 됩니다.
수행 방식 정의 예시
── 시니어가 "주니어 역할" 정의 ── ■ 코드 구현 시 - 설계 문서를 먼저 읽고, 설계에 없는 판단은 하지 말 것 - 구현 완료 후 수행결과 보고서 작성 - 보고서: 구현 범위, 변경 파일, 테스트 결과, 미해결 이슈 ■ 테스트 시 - Jest 단위 테스트, 커버리지 80%+ - 기존 테스트 깨지면 즉시 보고
── 주니어가 "선임 역할" 정의 ── ■ 요구사항 리뷰 시 - 빠진 항목·모순을 질문 형태로 정리 - "이렇게 해"가 아닌 "이건 고려했는가?" ■ 코드 리뷰 시 - 설계 대비 일치 여부 확인 - 개선점은 이유와 함께 제시 (학습)
